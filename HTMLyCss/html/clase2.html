Introducción a HTML Semántico

HTML semántico significa usar elementos HTML que describen su significado de manera clara para el navegador y el desarrollador. Esto mejora la accesibilidad y el SEO.

Beneficios:

  • Mejor accesibilidad
  • SEO mejorado
  • Código más legible
  • Estructura más clara

Sin semántica:

<div id="header">...</div>
<div class="nav">...</div>
<div class="main">...</div>

Etiquetas Semánticas

<header>

Representa el encabezado de una sección o documento. Contiene títulos, logos, navegación, etc.

<header>
  <h1>Mi Sitio Web</h1>
  <nav>...</nav>
</header>

<nav>

Define una sección con enlaces de navegación. No todos los grupos de enlaces necesitan estar en un <nav>.

<nav>
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Contacto</a></li>
  </ul>
</nav>

<section>

Define una sección temática en un documento. Debe tener un encabezado (h1-h6).

<section>
  <h2>Sobre Nosotros</h2>
  <p>Contenido...</p>
</section>

<article>

Contenido independiente que tendría sentido por sí solo (noticias, posts, comentarios).

<article>
  <h2>Título del Artículo</h2>
  <p>Contenido...</p>
</article>

<footer>

Pie de página para una sección o documento. Suele contener información de copyright, enlaces legales, etc.

<footer>
  <p>© 2023 Mi Sitio</p>
  <address>contacto@misitio.com</address>
</footer>

Estructura Completa

Cómo se combinan estas etiquetas en un documento típico.

<body>
  <header>...</header>
  <nav>...</nav>
  <main>
    <article>
      <section>...</section>
    </article>
  </main>
  <footer>...</footer>
</body>

Listas en HTML

<ul> - Listas Desordenadas

Se usan cuando el orden de los elementos no es importante. Se muestran con viñetas por defecto.

Código:

<ul>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
</ul>

Resultado:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

Ejemplo práctico:

<nav>
  <ul class="menu">
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Servicios</a></li>
    <li><a href="#">Contacto</a></li>
  </ul>
</nav>

<ol> - Listas Ordenadas

Se usan cuando el orden de los elementos es importante. Se muestran con números por defecto.

Código:

<ol>
  <li>Calentar agua</li>
  <li>Agregar café</li>
  <li>Colar y servir</li>
</ol>

Resultado:

  1. Calentar agua
  2. Agregar café
  3. Colar y servir

Atributos útiles:

  • start="n": Comienza la numeración desde n
  • reversed: Orden inverso (5, 4, 3...)
  • type="A/a/I/i/1": Tipo de numeración
<ol type="I" start="3">
  <li>Tercer lugar</li>
  <li>Cuarto lugar</li>
</ol>

Ejemplos Combinados

Estructura de Blog

<article class="post">
  <header>
    <h2>Título del Post</h2>
    <p class="meta">Publicado el <time>2023-05-15</time> por Autor</p>
  </header>
  <section class="content">
    <p>Contenido principal del artículo...</p>
    <ul>
      <li>Punto importante 1</li>
      <li>Punto importante 2</li>
    </ul>
  </section>
  <footer class="post-footer">
    <nav class="tags">
      <ul>
        <li><a href="#">#html</a></li>
        <li><a href="#">#web</a></li>
      </ul>
    </nav>
  </footer>
</article>

Receta de Cocina

<article class="recipe">
  <header>
    <h1>Tarta de Manzana</h1>
    <p>Tiempo: 60 minutos</p>
  </header>
  <section class="ingredients">
    <h2>Ingredientes</h2>
    <ul>
      <li>3 manzanas</li>
      <li>200g de harina</li>
      <li>100g de azúcar</li>
    </ul>
  </section>
  <section class="steps">
    <h2>Preparación</h2>
    <ol>
      <li>Pelar y cortar las manzanas</li>
      <li>Mezclar harina y azúcar</li>
      <li>Hornear a 180° por 30 min</li>
    </ol>
  </section>
  <footer>
    <p>Receta de la abuela</p>
  </footer>
</article>

Ejercicios Prácticos

Ejercicio 1: Estructura Semántica

Crea la estructura HTML semántica para una página de noticias con:

  • Un encabezado con título y subtítulo
  • Menú de navegación con 3 secciones
  • Área principal con 2 artículos de noticias
  • Cada artículo debe tener título, fecha y contenido
  • Un pie de página con información de copyright
<body>
  <header>
    <h1>El Periódico Digital</h1>
    <p>Las noticias más recientes</p>
  </header>
  <nav>
    <ul>
      <li><a href="#">Política</a></li>
      <li><a href="#">Deportes</a></li>
      <li><a href="#">Cultura</a></li>
    </ul>
  </nav>
  <main>
    <article>
      <h2>Nueva ley de educación</h2>
      <p><time>2023-05-10</time></p>
      <p>Contenido de la noticia...</p>
    </article>
    <article>
      <h2>El equipo local gana el campeonato</h2>
      <p><time>2023-05-09</time></p>
      <p>Contenido deportivo...</p>
    </article>
  </main>
  <footer>
    <p>© 2023 El Periódico Digital - Todos los derechos reservados</p>
  </footer>
</body>
