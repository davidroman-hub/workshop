Conceptos Básicos de HTML

Aprende los fundamentos del lenguaje de marcado web

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web mediante etiquetas que definen el contenido como encabezados, párrafos, imágenes, etc.

No es un lenguaje de programación, sino un lenguaje de marcado que le dice al navegador cómo mostrar el contenido.

HTML es como el esqueleto de una página web, mientras que CSS es la piel y JavaScript los músculos.

Características clave:

  • Define la estructura del contenido
  • Usa etiquetas para marcar elementos
  • Se combina con CSS y JavaScript
  • Es interpretado por navegadores web

Estructura de un documento HTML

Estructura básica: 

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Mi Página</title>
    </head>
    <body>
        <h1>Título Principal</h1>
        <p>Este es un párrafo de ejemplo.</p>
    </body>
</html>

DOCTYPE

Declara el tipo de documento y versión de HTML (HTML5 en este caso).

Elemento <html>

El contenedor raíz que engloba toda la página. El atributo "lang" define el idioma.

Sección <head>

Contiene metadatos, enlaces a CSS/JS, título de la página y otra información no visible.

Elementos <meta>

Proporcionan metadatos como el conjunto de caracteres y la configuración de la vista.

Elemento <title>

Define el título de la página que aparece en la pestaña del navegador.

Sección <body>

Contiene todo el contenido visible de la página web (texto, imágenes, etc.).

Etiquetas Básicas

Etiqueta Descripción Ejemplo
<html> Elemento raíz que contiene todo el documento HTML. <html>...</html>
<head> Contiene metadatos e información sobre el documento. <head>...</head>
<body> Contiene el contenido visible de la página web. <body>...</body>
<h1> a <h6> Encabezados de diferentes niveles (h1 es el más importante). <h1>Título</h1>
<h2>Subtítulo</h2>
<p> Define un párrafo de texto. <p>Este es un párrafo.</p>

Este es un párrafo normal. HTML permite estructurar el contenido en párrafos para mejorar la legibilidad.

Cada párrafo se coloca dentro de las etiquetas <p> y </p>. Los navegadores automáticamente añaden un margen después de cada párrafo.

Práctica: Crea tu primera página HTML

Instrucciones:

  1. Crea un nuevo archivo con extensión .html
  2. Escribe la estructura básica de HTML5
  3. Añade un título principal con <h1>
  4. Incluye al menos dos párrafos con <p>
  5. Guarda el archivo y ábrelo en tu navegador

Consejo

Puedes usar editores como VS Code, Sublime Text o incluso el Bloc de notas para crear tu archivo HTML.

Desafío adicional:

Una vez que hayas creado tu primera página, intenta:

  • 1 Añadir más párrafos con información sobre ti
  • 2 Incluir un subtítulo con <h2>
  • 3 Cambiar el texto dentro de <title> y ver cómo afecta a la pestaña del navegador
  • 4 Experimentar con diferentes niveles de encabezados (h1-h6)
