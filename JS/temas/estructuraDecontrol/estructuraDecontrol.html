Ejercicios JavaScript Interactivos

Determinar Mayoría de Edad

Este ejercicio evalúa si una persona es mayor o menor de edad basado en un valor numérico.

Resultado:

Esperando entrada...

Código JavaScript:

let edad = parseInt(20);
if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}

Determinar Día de la Semana

Este ejercicio muestra el nombre del día de la semana basado en un número del 1 al 7.

Resultado:

Esperando entrada...

Código JavaScript:

let dia = parseInt("20");
switch (dia) {
    case 1: console.log("Domingo"); break;
    case 2: console.log("Lunes"); break;
    case 3: console.log("Martes"); break;
    case 4: console.log("Miércoles"); break;
    case 5: console.log("Jueves"); break;
    case 6: console.log("Viernes"); break;
    case 7: console.log("Sábado"); break;
    default: console.log("Número inválido"); break;
}

Explicación de los Ejercicios

Condicional if-else

El primer ejercicio utiliza una estructura condicional if-else para determinar si una persona es mayor o menor de edad.

  • parseInt() convierte el valor a número entero
  • edad >= 18 es la condición que se evalúa
  • Si es verdadera, ejecuta el primer bloque
  • Si es falsa, ejecuta el bloque después del else

Estructura switch

El segundo ejercicio usa una estructura switch para seleccionar entre múltiples casos basados en el valor de una variable.

  • Evalúa una expresión y la compara con cada case
  • Cuando encuentra coincidencia, ejecuta ese bloque
  • break evita que se ejecuten los casos siguientes
  • default se ejecuta si no hay coincidencias
